# Miniconda 工具

## 常用命令

### 创建虚拟环境

```bash
conda create --name [env_name] python=[x.y.z] -y
```

### 激活虚拟环境

```bash
conda activate [env_name]
```

### 退出当前虚拟环境

```bash
conda deactivate
```

### 删除虚拟环境

```bash
conda env remove 【env_name】
```
